# Dr. Haydee Laza's Lab — Texas Tech University

**Department of Plant and Soil Science**

Research tools and data pipelines developed by Aarti Khatri for ongoing soil science research, including gas exchange analysis, nutrient management, and greenhouse experiment support.

### Projects

- **[licor-li7815-analysis](https://github.com/ttu-laza-lab/licor-li7815-analysis)** — Python pipeline for processing LI-7815 cavity ring-down spectroscopy data, computing CO₂ and H₂O flux for gas exchange analysis.
- **[fertilizer-rate-calculator](https://github.com/ttu-laza-lab/fertilizer-rate-calculator)** — Web-based fertilizer application rate calculator for greenhouse cone experiments. Live at [egrow-calculator.vercel.app](https://egrow-calculator.vercel.app).
